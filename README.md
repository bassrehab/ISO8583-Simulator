# ISO8583 Simulator

A hybrid ISO 8583 message simulator with both CLI and web interface capabilities.

## Features

- ISO 8583 message parsing and building
- Command-line interface for automation
- Web interface for interactive testing
- Configurable message templates
- Comprehensive validation
- Test report generation

## Installation

```bash
pip install iso8583sim
```

## Quick Start

### CLI Usage

```bash
iso8583sim --help
```

### Web Interface

```bash
iso8583sim web
```
